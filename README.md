# Smart Wind Turbine Failure Detection System
**Graduation Thesis Project | Dokuz Eylül University**

## 📌 Project Overview
This project focuses on real-time failure diagnosis for wind turbines using an end-to-end Machine Learning pipeline integrated with IoT infrastructure. The system aims to minimize maintenance costs and downtime by predicting potential faults before they occur.

## 🛠 Tech Stack
- **Languages:** Python, SQL
- **Machine Learning:** XGBoost, Random Forest, SHAP, LIME (Explainable AI)
- **Generative AI:** RAG (Retrieval-Augmented Generation), LLM integration for diagnostic reports
- **Backend & IoT:** Oracle APEX, Arduino/ESP32 Sensors
- **Data Handling:** SMOTE (for class imbalance)

## 📊 Key Performance Metrics
- **Accuracy:** 89.7%
- **ROC-AUC:** 95.7%
- **F1-Score:** 90.1%

## 🔍 Key Features
- **Real-time Monitoring:** Continuous data acquisition from IoT sensors.
- **Explainable AI (XAI):** Utilizing **SHAP** and **LIME** to interpret model predictions, ensuring transparency for engineers.
- **Automated Reporting:** Integrated LLM to generate human-readable maintenance reports based on predicted fault types.
